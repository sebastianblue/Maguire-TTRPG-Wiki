## Supported Formats
- `.mp3` (recommended - best browser compatibility)
- `.wav` (higher quality but larger files)
- `.ogg` (good compression, wide support)
- `.m4a` (Apple devices)

## File Size Recommendations
- **Ambient loops**: 3-10 MB each (3-5 minute loops)
- **Music box melodies**: < 1 MB each (30-60 seconds)
- **Sound effects**: < 500 KB each (1-10 seconds)
- **Combat themes**: 5-15 MB each (3-5 minute loops)
