## Usage Guidelines

### Converting Templates to HTML

1. Replace markdown headers with HTML header tags
2. Convert tables to HTML table format
3. Convert lists to `<ul>` and `<li>` tags
4. Wrap paragraphs in `<p>` tags
5. Convert blockquotes to `<blockquote>` tags
6. Make links functional with `onclick="Tabs.openTab('page-id', 'Page Title'); return false;"`

### Internal Linking

Always use this format for links between pages:
```html
<a href="#" onclick="Tabs.openTab('page/path/id', 'Page Title'); return false;">Link Text</a>
```

### Maintaining Consistency

- Always include the signature quote at the top
- Follow the exact section order
- Use the same heading levels
- Include "Related Entries" section at the end
- Tag all pages appropriately in pages.json

### Template Tags

Use these tags in pages.json:
- `template` - Marks this as a template page
- `race`, `class`, `creature`, `location`, `spell`, `background`, `magic-item`, `faction`, `npc`, `condition` - Content type
- Specific subtags like `human`, `construct`, `gillikin`, `grimmerie`, etc.
