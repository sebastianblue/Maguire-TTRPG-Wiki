## Printing Tips

### Standard Print
Most files are designed for standard 8.5" x 11" paper:
- 01, 02, 03, 04, 05, 10, 11, 12

### Special Print
- **06 (Name Cards)** — Print on cardstock if possible, cut into 8 cards
- **07 (I LOVE U)** — Can print on darker paper for dramatic effect
- **08 (Cabinet Labels)** — Print on gold/brass colored paper if available, or use yellow cardstock
- **09 (Telescope Note)** — Print small, fold up, hide inside telescope

### Fonts Used
The handouts use these fonts (already included in your project):
- Normal Handwriting Font.woff
- MattB Regular.ttf (handwriting)
- Milonguita (Script Cursive Font).ttf

**Note:** If fonts don't display when printing, the files will fall back to Courier New and Georgia.

## Converting to PDF

### Option 1: Print to PDF from Browser
1. Open each .html file in Chrome/Firefox/Safari
2. File → Print
3. Select "Save as PDF" as the destination
4. Save to this folder with same name (.pdf extension)

### Option 2: Use wkhtmltopdf (if installed)
```bash
cd handouts-pdf
for file in *.html; do wkhtmltopdf "$file" "${file%.html}.pdf"; done
```

### Option 3: Use your browser's developer tools
Most browsers can print HTML files directly to PDF with good formatting.

## Organization Tips

### Physical Organization
1. **Seal Props Bags** — Put each seal's props in a labeled plastic bag:
   - Seal 1: 4 music boxes + sheet music handout
   - Seal 2: 8 fabric swatches
   - Seal 3: 4 books + puzzle pieces
   - Seal 4: Deck of cards + daily rounds handout
   - Seal 5: Telescope + matches + note + REAL CANDLE
   - Seal 6: Masquerade mask
   - Seal 7: 8 name cards (pre-placed at table)

2. **Cabinet Box** — Put all 8 assembled cabinets in one box
   - Each cabinet should have its label attached
   - Each cabinet should contain its loot cards and any story handouts

3. **Reward Cards Box** — Keep cut-out reward cards in envelopes:
   - Envelope 1: Seal reward cards (7 cards)
   - Envelope 2: Cabinet loot cards (15 cards)
