## Quick Start for GMs

1. **Read**: `the_witchs_wake_complete_v2.md`
2. **Print**: Handouts from `/handouts-pdf/` (see HANDOUTS-SUMMARY.md)
3. **Prepare**: Props using `handouts-pdf/11-props-checklist.html`
4. **Run**: Use `/dashboard.html` during session

## Win Condition

Players must:
1. Break all 7 Seals → Unlocks Cabinet 8
2. Open all 8 Cabinets → Get loot + Ring of Spell Storing (Dispel Magic 5th level)
3. Kill Yackle → Use Dispel Magic from Cabinet 8 to destroy her phylactery-heart
4. Survive until dawn → Escape the estate
