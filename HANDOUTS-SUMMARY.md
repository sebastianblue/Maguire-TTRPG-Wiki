### Option 1: Open in Browser & Print to PDF
1. Open each .html file in Chrome, Firefox, or Safari
2. Press **Ctrl+P** (or Cmd+P on Mac)
3. Select "Save as PDF" as printer/destination
4. Save to `handouts-pdf/` folder with .pdf extension
5. Print PDFs as needed

### Option 2: Print Directly from HTML
- Modern browsers render these HTML files perfectly for printing
- Fonts are embedded or fall back gracefully
- Background colors print correctly

- **Color:** Yes (for visual impact, especially handout 07)
- **Paper:** Standard 8.5" x 11"
